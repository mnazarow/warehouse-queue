Инструкция по настройке 1С

1. Получение заказов
GET /api/integration/1c/orders?token=ВАШ_ТОКЕН

Параметры запроса:
date — заказы на конкретную дату (ГГГГ-ММ-ДД)
date_from, date_to — период
status — фильтр: pending, confirmed, in_progress, completed

2. Изменение статуса заказа
POST /api/integration/1c/orders/ID/status?token=ВАШ_ТОКЕН
Тело: {"status": "complete"} или {"status": "cancel"}

Токен передаётся через query-параметр token или заголовок X-API-Token.

Логотипы

Загрузите логотип для каждой темы. Рекомендуемый размер: 200×60px (PNG с прозрачностью). Если логотип для темы не задан, используется стандартный /logo.png.

Redis

TTL кэша Redis

Время жизни кэша по типам данных, в секундах. 0 — кэшировать без срока. Действует при включённом Redis; после сохранения кэш сбрасывается.

PostgreSQL

Миграция переносит все данные из SQLite в PostgreSQL или обратно. Таблицы в PostgreSQL будут созданы автоматически.

Переключение меняет активную базу данных на лету без перезапуска сервера.
